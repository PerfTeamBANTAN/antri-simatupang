# Antri Simatupang - Butik Emas LM ANTAM

Website sistem pendaftaran antrian untuk Butik Emas LM ANTAM di TB Simatupang, Jakarta Selatan. Duplikasi website dari https://antrisimatupang.com/

## Fitur Utama
- Informasi pendaftaran antrian pembelian emas
- Cek status antrian secara real-time
- Informasi rencana ketersediaan stok
- Bukti transfer pembelian
- Interface responsif untuk desktop dan mobile

## Jam Operasional
- **Layanan Pembelian**: Senin - Jumat
- **Jam Operasional**: 08:30 WIB - Selesai
- **Waktu Pendaftaran**: H-1 pkl. 15:00 - 16:00 WIB (hari kerja)
- **TUTUP**: Sabtu, Minggu, Libur Nasional

## Informasi Penting
- Stok dapat berubah sewaktu-waktu
- Pendaftaran antrian tidak menjamin ketersediaan produk
- Pembagian nomor antrian dimulai pkl. 08:30 WIB
- Tidak hadir saat pembagian harus mengambil nomor antrian baru
- Verifikasi KTP wajib dilakukan

## Alternatif Pembelian
Saat BELM tutup, pembelian emas masih dapat dilakukan melalui:
- www.logammulia.com
- Butik Emas ANTAM Official Store (e-commerce)
- Livin Sukha Platform

## Cara Mengakses
1. Buka browser dan akses halaman website
2. Pilih menu "Daftar Antrian" untuk pendaftaran
3. Masukkan data yang diperlukan
4. Dapatkan kode antrian
5. Tunggu pengumuman ketersediaan stok
